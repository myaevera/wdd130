Cabin Trip Site Plan

Mya Vera

WDD 130-30

Overview

Purpose

Create a decriptive website that will help my friends get all the details for our cabin trip so that they don't have to text me whenever they have questions.

Audience

Hey Everyone!! Thanks for clicking on the link. I hope you are all excited for this trip. I figured since we are all spread out it would be easiest to have one place with all the information you will need for the trip. This website will feature pictures of the place we want to stay at, fun things to do in the area and the last thing we all want to think about... MONEY. If you have any questions regarding the trip please refer to this site first before contacting someone. If you can't find what you are looking for on here then contact someone.I'm so excited for this trip and hope you all can come! Please RSVP!!
